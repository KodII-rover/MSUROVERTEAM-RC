# ПРОГРАММНОЕ ОБЕСПЕЧНЕИЕ ДЛЯ ДИСТАНЦИОННОГО УПРАЛВЕНИЯ РОВЕРОМ

## MSUROVERTEAM-RC

Движение ровера и его операции во время миссий должны контролироваться оператором, в т.ч. при выполнении автономной миссии. Для этого разработано Программное обеспечение для дистанционного управления ровером (ПО).\
Данное ПО не является предметом договора на разработку открытых библиотек по программе **"КодИИ"**.

Это стандартное Web приложение, для работы которого необходим Web-сервер, запущенный на ровере в ROS2 Humble, и любой современный Web-браузер на утсройстве оператора. Устройство оператора должно быть подключено к сети ровера.\
ПО не зависит от платформы и работает под управлением любого современного интернет-браузера. В ПО реализован интерфейс для управления движением и видеонаблюдением в режиме реального времени, которые обеспечивает:

*   удобное задание команд движения,
*   визуальный контроль через потоки с камер,
*   стабильную работу даже в условиях нестабильной сети и сложного рельефа.
