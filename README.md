# ПРОГРАММНОЕ ОБЕСПЕЧНЕИЕ ДЛЯ ДИСТАНЦИОННОГО УПРАЛВЕНИЯ РОВЕРОМ

## MSUROVERTEAM-RC

Движение ровера и его операции во время миссий должны контролироваться оператором, в т.ч. при выполнении автономной миссии. Для этого разработано Программное обеспечение для дистанционного управления ровером (ПО). Данное ПО не является предметом договора на разработку открытых библиотек по программе **"КодИИ"**.

Это стандартное Web приложение, для работы которого необходим Web-сервер, запущенный на ровере под управлением ROS2 Humble, и любой современный интернет-браузер на утсройстве оператора. Устройство оператора должно быть подключено к сети ровера. В ПО реализован интерфейс для управления движением ровера и видеонаблюдением в режиме реального времени, которые обеспечивает:

*   удобное задание команд движения,
*   визуальный контроль через видеопотоки с камер,
*   стабильную работу даже в условиях нестабильной сети и сложного рельефа.

За основу при разработке ПО дистанционного управления была взята бета-версия пакета [turtlebro_web](https://github.com/voltbro/turtlebro_web), разработанная нашими коллегами из НИИ Механики МГУ на основе стандартной ROS2 библиотеки roslibjs.

## Сборка и запуск Web-интерфейса
```bash																																																										
# Сборка пакета
colcon build --symlink-install --packages-select turtlebro_web

# Запуск Web-сервера через launch
ros2 launch turtlebro_web web_server.xml

# Подготовка зависимостей
rosdep install -i --from-path src --rosdistro humble -y
